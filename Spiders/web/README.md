# 股票信号分析系统 Web 客户端

这是一个用于展示股票信号分析结果的 Web 应用。

## 功能特性

- 📊 数据统计展示（总信号数、平均成功率、股票数量等）
- 🔍 多条件筛选（股票代码、名称、信号类型、成功率等）
- 📈 数据表格展示（支持排序、分页）
- 🎨 现代化美观界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python app.py
```

然后在浏览器中访问：http://localhost:5001

## 使用说明

1. **查看统计信息**：页面顶部显示总体统计数据
2. **筛选数据**：使用筛选条件区域设置筛选条件，点击"搜索"按钮
3. **排序数据**：点击表头可以按该列排序
4. **分页浏览**：使用底部分页控件浏览更多数据

## API 接口

- `GET /api/signals` - 获取信号数据（支持筛选和分页）
- `GET /api/stats` - 获取统计数据
